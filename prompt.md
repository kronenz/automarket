## 프로젝트 개요

이커머스/브랜드 사업의 전체 라이프사이클을 AI Agent 기반 그래프 파이프라인으로 관리하는 플랫폼.
각 노드가 독립적인 SaaS 도구로 가치가 있으면서, 점진적으로 통합되어 전체 파이프라인을 완성하는 구조.

### 설계 철학 — 제1원리(First Principles) 기반

1. **근본 분해**: 이커머스에서 절대적으로 필요한 것은 3가지뿐 — Supply(팔 물건), Demand(살 사람), Match(둘의 연결)
2. **의사결정 단위 기준**: 노드를 "부서"가 아닌 "의사결정 단위"로 구성
3. **AI-네이티브**: 사람의 조직 구조가 아닌 AI Agent의 실행 단위로 재설계
4. **속도 최적화**: Time-to-Revenue (트렌드 감지 → 첫 매출)를 24시간 이내로 목표
5. **아키텍처 진화**: v2(9노드, 프로세스 복사) → v3(5노드, 제1원리 압축) → v5(6노드, +소셜프루프) → v6(4노드, 최종 아키텍처)

### v6 최종 아키텍처 — 4 Core Nodes

**Node 1: 기회 발견 엔진 (What to sell) — AI 92%**
- 역할: 시장 신호 수집 → 상품 후보 추출 → 수익성 검증을 하나의 루프로 통합
- v2에서 통합된 영역: 인텔리전스 + 소싱 & 상품기획 + 정산 & 수익분석
- 내장 신경계: 트렌드 스코어 실시간 갱신, 마진율 변동 자동 재계산, 경쟁 진입 감지 → 기회 스코어 자동 하향
- TTR 목표: 트렌드 감지 → 상품 확정 4시간
- 태스크:
  - Autopilot: 트렌드 크롤링 & 수요 예측, 경쟁사 모니터링 & 포지션 맵, 마진 시뮬레이션 & 수익성 검증, 공급처 자동 매칭 & 견적 비교
  - Copilot: 상품 포트폴리오 추천 (AI가 Top N 추천 → 사업자 선택)
  - Human-led: Go/No-go & 공급처 최종 계약

**Node 2: 고객 인텔리전스 (Who to sell to) — AI 82%**
- 역할: 고객 데이터 + 페르소나 + 브랜드 전략 통합. 브랜딩은 "고객을 이해한 결과물"
- v2에서 통합된 영역: 고객 & CRM + 브랜딩 & 전략
- 내장 신경계: 구매 전환율 변동 → 페르소나 자동 재조정, 이탈율 급증 → 세그먼트 자동 리밸런싱, LTV 변동 → 가격 전략 파라미터 자동 갱신
- TTR 목표: 고객 신호 → 타겟 조정 실시간
- 태스크:
  - Autopilot: 행동 데이터 클러스터링, LTV 예측 & 이탈 조기경보, 페르소나 자동 생성/갱신
  - Copilot: 타겟 세그먼트 최적화
  - Human-led: 브랜드 포지셔닝/톤앤매너 (초기 1회 사람 설정 → 이후 AI 일관성 유지)

**Node 3: 콘텐츠 공급망 (How to present & trust) — AI 78%**
- 역할: 내부 생산(AI Generator)과 외부 생산(Partner Network)을 하나의 공급망으로 통합
- v2에서 통합된 영역: 콘텐츠 생산 + 소셜 프루프 엔진(리뷰/인플루언서)
- 듀얼 엔진 구조:
  - AI Generator (내부): 상세페이지 자동 생성, SNS 멀티포맷 변환, 광고 소재 배리에이션, A/B 카피 자동 생성
  - Partner Network (외부): 리뷰어 DB & 체험단 운영, 협력사 리뷰 발주 관리(견적/계약/납품), 인플루언서 매칭 & 트래킹, UGC 수집 & 큐레이션
- 내장 신경계: 클릭률 하락 → 소재 자동 교체, 리뷰 감성 악화 → 체험단 자동 증원, 인플루언서 ROI 저하 → 자동 예산 리밸런싱
- TTR 목표: 상품 확정 → 콘텐츠 완성 2시간
- 태스크:
  - Autopilot: 상세페이지/광고소재 AI 생성, SNS 멀티채널 자동 배포, 리뷰어 DB 관리 & 체험단 모집, 협력사 견적 비교 & 발주 관리, 리뷰 품질 스코어링
  - Copilot: 인플루언서 후보 추천, 크리에이티브 디렉션 검수, 캠페인 전략 설계, 법적 표현 확인 (AI 1차 필터 → 위험 건만 사람)
  - Human-led: 인플루언서 핵심 파트너십 협상

**Node 4: 채널 오케스트레이터 (Where to reach) — AI 85%**
- 역할: 유통과 마케팅을 하나로 통합한 실행 엔진. 가격, 광고, 재고가 실시간으로 서로 조절
- v2에서 통합된 영역: 유통 & 채널 + 마케팅 자동화
- 내장 신경계: ROAS 하락 → 예산 자동 리밸런싱, 재고 부족 → 광고 자동 중단, 채널별 전환율 변동 → 가격 자동 조정
- TTR 목표: 콘텐츠 완성 → 채널 라이브 1시간
- 태스크:
  - Autopilot: 멀티채널 가격 실시간 최적화, 광고 예산 자동 배분, 재고↔광고↔가격 연동, 채널별 성과 리포트 자동 생성
  - Copilot: 프로모션 전략 제안
  - Human-led: 신규 채널 진입 / 전략적 제휴

### 핵심 설계 요소

**성과 피드백 = 독립 노드가 아닌 "신경계(Nervous System)"**
- 모든 노드에 내장된 실시간 피드백 루프
- 각 노드가 자기 성과를 자동 감지하고 파라미터를 조정
- 중앙 대시보드는 "사람이 보는 것"이지 "의사결정 노드"가 아님

**3단계 자동화 레벨 + Trust Escalation**
- Autopilot (🤖): AI 자율 실행 → 사후 보고 (약 60%)
- Copilot (🤝): AI 제안 → 사람 승인 (약 25%)
- Human-led (👤): 사람 결정 → AI 보조 (약 15%)
- 승격 조건: 연속 N회 사람과 동일 판단 → Copilot에서 Autopilot으로 승격
- 롤백 조건: 손실 임계값 초과 시 즉시 강등

**사람이 해야 하는 3가지 의사결정**
1. 사업 방향 결정: 뭘 할 것인가, 계속할 것인가, 피벗할 것인가
2. 관계 구축: 핵심 파트너와의 신뢰와 장기적 관계
3. 위기 대응: 예측 불가능한 상황에서의 판단

### 12개 독립 SaaS 제품 상세

**기회 발견 엔진 → 3개 SaaS**

1. TrendRadar (트렌드 레이더)
   - 타겟: 이커머스 셀러, MD, 상품기획자
   - 핵심 기능: 실시간 트렌드 크롤링, 키워드 수요 예측 AI, 카테고리별 기회 스코어링, 경쟁 강도 분석
   - 단독 가치: "지금 뭐가 뜨고 있는지" AI 수요 예측
   - 해자: 네이버/쿠팡/SNS 공개 데이터로 자급자족
   - 경쟁 참고: 셀러마스터, 아이템스카우트 (키워드 중심 → AI 수요 예측으로 차별화)

2. SourceMatch (소스매치)
   - 타겟: 소싱이 필요한 셀러, 브랜드
   - 핵심 기능: 공급처 DB & 자동 매칭, 조건 비교표 자동 생성, MOQ/리드타임 계산, 견적 요청 자동화
   - 단독 가치: 공급처 탐색 시간 90% 절감
   - 해자: 공급처 DB 자체 구축 (네트워크 효과)
   - 통합 시: TrendRadar 기회 스코어 → SourceMatch 자동 매칭

3. MarginSim (마진심)
   - 타겟: 셀러, 브랜드, 위탁판매자
   - 핵심 기능: 원가/물류/수수료/광고비 통합 마진 시뮬레이터, 채널별 수익성 비교, 시나리오 분석, Go/No-go 자동 판단
   - 단독 가치: "이거 팔면 진짜 남나?" 즉답
   - 해자: 채널 수수료 DB + 사용자 축적 데이터
   - 통합 시: TrendRadar → SourceMatch → MarginSim 자동 Go/No-go

**고객 인텔리전스 → 3개 SaaS**

4. PersonaAI (페르소나AI)
   - 타겟: 마케터, 브랜드 매니저, 1인 셀러
   - 핵심 기능: 구매 데이터 기반 자동 페르소나 생성, 세그먼트 빌더, 타겟 추천, 코호트 분석
   - 단독 가치: 데이터 기반 고객 이해 자동화
   - 해자: 스마트스토어/쿠팡/자사몰 주문 데이터 연동
   - 경쟁 참고: 채널톡 등 기존 CRM은 CS 중심 → "판매 전략을 위한" 고객 분석

5. BrandForge (브랜드포지)
   - 타겟: 신규 브랜드, D2C 스타트업
   - 핵심 기능: AI 브랜드 가이드 생성(톤앤매너, 컬러, 메시징), 포지셔닝 맵, 경쟁 브랜드 대비 분석
   - 단독 가치: 브랜딩 에이전시 비용을 AI로 대체
   - 해자: 경쟁사 공개 데이터 + 사용자 입력
   - 통합 시: PersonaAI 세그먼트 → BrandForge 자동 포지셔닝

6. ChurnGuard (이탈방지)
   - 타겟: 자사몰/D2C 운영자, 구독 서비스
   - 핵심 기능: LTV 예측 모델, 이탈 조기경보, 재구매 유도 자동화, 코호트 분석
   - 단독 가치: 이탈 고객 사전 포착 & 방지
   - 해자: 주문/방문 데이터 연동
   - 통합 시: 이탈 신호 → PersonaAI 세그먼트 자동 갱신

**콘텐츠 공급망 → 4개 SaaS**

7. PageFactory (페이지팩토리)
   - 타겟: 모든 이커머스 셀러
   - 핵심 기능: 상품 정보 입력 → 상세페이지 AI 자동 생성, 템플릿 라이브러리, A/B 버전 자동 생성, 채널별 최적화
   - 단독 가치: 상세페이지 외주 50~200만원 → AI 10분. 가장 즉각적인 PMF 후보
   - 해자: 이커머스 상세페이지 전문 AI 생성 엔진
   - 경쟁 참고: 미리캔버스/캔바 (범용 디자인) → 이커머스 상세페이지 전문

8. SocialSpray (소셜스프레이)
   - 타겟: SNS 마케팅하는 셀러, 브랜드
   - 핵심 기능: 원소스 멀티포맷 자동 변환(인스타/틱톡/블로그/유튜브 숏츠), 예약 배포, 성과 트래킹
   - 단독 가치: 멀티채널 콘텐츠 원클릭
   - 해자: 채널 API 연동 생태계
   - 통합 시: PageFactory 에셋 → SocialSpray 자동 배포

9. ReviewOps (리뷰옵스)
   - 타겟: 체험단/리뷰 운영하는 셀러, 마케팅 대행사
   - 핵심 기능: 리뷰어 DB & 체험단 모집→배송→작성 관리 자동화, 협력사 리뷰 발주 관리(견적/계약/납품 트래킹), 리뷰 품질 스코어링
   - 단독 가치: 리뷰 운영 전체 자동화 (현재 시장 공백)
   - 해자: 리뷰어 DB 네트워크 효과 (쌓일수록 해자)
   - 통합 시: 가장 강력한 네트워크 효과

10. InfluBridge (인플루브릿지)
    - 타겟: 인플루언서 마케팅하는 브랜드, 대행사
    - 핵심 기능: AI 인플루언서 매칭(카테고리/팔로워/참여율), 캠페인 관리, ROI 트래킹, 장기 파트너십 관리
    - 단독 가치: 인플루언서 발굴→관리→ROI 풀스택
    - 해자: SNS 크롤링 + 캠페인 성과 DB
    - 통합 시: ReviewOps + InfluBridge = 소셜 프루프 풀스택

**채널 오케스트레이터 → 2개 SaaS**

11. ChannelSync (채널싱크)
    - 타겟: 멀티채널 셀러
    - 핵심 기능: 쿠팡/스마트스토어/자사몰/11번가 통합 관리, 재고 동기화, 주문 통합, 채널별 성과 비교
    - 단독 가치: 채널별 따로 관리 → 하나로 통합
    - 해자: 각 채널 API 연동 깊이
    - 경쟁 참고: 사방넷/플레이오토 (주문/재고 중심) → 성과 분석까지 포함

12. AdPilot (애드파일럿)
    - 타겟: 퍼포먼스 마케팅하는 셀러, 브랜드
    - 핵심 기능: 광고 예산 자동 배분(ROAS 기반), 재고 연동 자동 on/off, 크로스채널 성과 대시보드, 실시간 최적화
    - 단독 가치: 광고비 낭비 AI 자동 절감
    - 해자: 광고 플랫폼 API + 매출 데이터 축적
    - 통합 시: ChannelSync 재고 → AdPilot 자동 조절

### SaaS 독립화 기준

각 SaaS를 독립 제품으로 분리할 때 4가지 기준:
1. 단독 가치 테스트: "이것만 써도 월 N만원 낼 만한가?"
2. 타겟 명확성: "누가 이걸 쓰는가?"가 한 문장으로 설명되는가
3. 데이터 자급자족: 다른 노드 없이도 자체 데이터로 동작하는가
4. 통합 시 증폭: 연결하면 단독 대비 가치가 2배 이상 되는가

### 실행 로드맵

- Phase 1 (MVP): PageFactory, TrendRadar, ReviewOps — PMF 가장 빠른 3개
- Phase 2 (확장): SocialSpray, MarginSim, AdPilot — Phase 1 데이터 활용
- Phase 3 (심화): PersonaAI, BrandForge, InfluBridge — 고객 데이터 축적 후
